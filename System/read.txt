Чтобы скомпилировать системные замены, нужно выполнить:
	DCC32.EXE -q system SysSfIni SYSWSTR sysinit -m -y -z -$D-
либо:
	DCC32.EXE -q system sysinit -m -y -z -$D-
либо
	MAKE
	
D2006beta-D2009beta - это бета версии замен, они основаны на D7_avenger, возможны лаги
D2006_orig - чуть облегченная версия оригинальных модулей
